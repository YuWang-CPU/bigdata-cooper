# 命名实体识别CRF代码模块README
## 1.模块实现功能
	1.读入.txt、.doc、.xml等文件。
	2.将文件中内容与实体库进行精确匹配得到实体。
	3.根据规则对文本内容进行规则实体识别。
	4.根据CRF对文本内容进行实体识别。
